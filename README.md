# Small app "List of users"

> A Vue.js project. Use fake REST API, bootstrap

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

# to run json server

json-server --watch db.json --port 3004

